## Run Locally

Clone the project

```bash
  git clone https://github.com/krishnamaurya31/react-ecommerce.git
```

Go to the project directory

```bash
  cd react-ecommerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
